Around The Web

Il s’agit ici d’un projet tutoré réalisé en fin de 1ère année de DUT MMI. Le but de l’exercice était de créer une agence fictive de communication digitale. Il fallait développer le site web de cette agence, designer le logo, et faire que diverses réalisations prints associées (carte de visite, dépliant/flyer). Nous avons aussi réalisé une vidéo promotionnelle à l’aide d’After Effects.

Mot-clés :
HTML5 CSS3 Materialize Javascript Jquery

Grenoble Out

Grenoble Out a été réalisé dans le cadre du cours d’Algorithmique et développement web. L’objectif était de réaliser un site web utilisant, entre autres, du PHP et SQL.
Nous avons donc fait un site regroupant des idées de sortie sur Grenoble. Le site permet aussi de trier les sorties selon certains critères.
L’aspect esthétique n’était pas évalué. Le framework Materialize a été utilisé.

Pensée Jardin

Pensée Jardin est un site web réalisé en fin de premier semestre de 1ère année de DUT MMI. Ont été utilisés de l’HTML5 et de la CSS3 pour faire ce site. Il s’agit d’un site de conseils en jardinage avec des astuces et des infos sur diverses plantes.
